# 터미널과 셸이란?
터미널은 컴퓨터와 사용자간에 상호작용할 수 있게 연결하는 장치<br>
셸이란 CLI 프로그램으로 사용자가 운영체제에 명령을 시킬수 있게 해주는 프로그램<br>

## 셸 스크립트
셸에서 동작 가능한 명령을 모아놓은 파일을 셸 스크립트라함<br>
이를 활용해서 자동화 프로그램도 가능!<br>

## 명령어 기본 형식
```zsh
$ 명령어 [옵션] [인자]
```
형식에서 [] 대괄호는 선택사항을 의미함<br>
인자에는 명령어에 전달하는 값으로 보통 파일이름이나 디렉터리 이름이 들어감<br>

## man 메뉴얼 확인 명령어
```zsh
man [옵션] [섹션] 페이지
```
메뉴얼의 약자로 명령어나 개념 등에 대한 메뉴얼을 확인 할 수 있는 명령어<br>
페이지는 조회 대상을 뜻함<br>
섹션은 각 페이지를 기준에 맞게 분류해 놓은것임<br>
같은 이름의 페이지가 여러 섹션에 존재하기도 함<br>

## ls 목록 출력
```zsh
ls [옵션] [파일]
```
파일목록들을 모두 출력해주는 명령어<br>
주요 옵션들
-a : 모든 파일 출력<br>
-l : 파일의 여러속성을 포함해서 출력<br>
-t : 생성된 순서로 파일출력<br>
-R : 하위 모든 디렉토리 순회<br>
-h : 사람이 읽기 좋은 크기로 파일 출력<br>

파일에 이름이나 패턴을 입력하면 일치하는 파일들만 출력가능


## cd 디렉토리 변경 명령어
```zsh
cd [디렉토리]
```
현재 작업 디렉토리의 위치를 변경하는 명령어의 약자<br>
cd .. : 현재 디렉토리의 상위 디렉토리로 이동<br>
cd 다운로드/ : 다운로드 디렉토리로 이동


## pwd 디렉토리 확인
현재 작업 디렉토리의 절대 경로를 표기해주는 명령어<br>
절대 경로란 루트 디렉토리를 기준으로 찾는 경로를 말하며<br>
상대 경로는 지금 현재 디렉토리를 기준으로 찾는 경로를 뜻함<br>



## cat
파일의 내용을 연결하거나 조회할때 사용하는 명령어

```zsh
cat [옵션] [ 파일]
```

## exit
현재 실행중인 셸을 종료합니다.

```zsh
exit [종료코드]
```
프로세스는 실행 중인 프로그램을 뜻하며<br>
종료코드는 실행한 프로세스가 종료되면서 남기는 유언과 같은것<br>
exit 명령어를 사용하면 셸을 종료할때도 종료코드를 남길 수 있음


## nano 
텍스트 편집기를 사용할 수 있는 명령어
```zsh
nano [파일]
```
파일을 지정하면 기존 파일을 열거나 아니면 새로운 파일을 만들어냄
